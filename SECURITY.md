# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an e-mail to **<support@sitefiguraviva.com>**. All security vulnerabilities will be promptly addressed.

**SLA**: We aim to triage all reports within **72 hours**.

### Scope

- Application logic vulnerability
- Data leakage
- Infrastructure misconfiguration

### Out of Scope

- DMARC/SPF records
- Social Engineering
- DOS/DDOS
